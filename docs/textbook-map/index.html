第一单元 阅读：春天与成长

古诗二首找春天开满鲜花的小路邓小平爷爷植树语文园地一

核心知识点

  • 春天景物词语
  • 古诗背诵与诗句理解
  • 动词和形容词积累
  • 按顺序观察图画
  • 句子补充完整

复习检查

  • 会背会写古诗重点句
  • 能区分近义词和反义词
  • 写话时能写清看到什么、想到什么

细化考点

课文和背诵
  • 《村居》《咏柳》会背、会默写重点诗句
  • 理解“草长莺飞、拂堤杨柳、碧玉妆成、绿丝绦”等词语画面
  • 《找春天》抓住春天在哪里：小草、野花、嫩芽、小溪
  • 《开满鲜花的小路》能说清礼物从哪里来、最后变成什么
  • 《邓小平爷爷植树》按时间、地点、人物、事情复述
字词句
  • 会写“诗、村、童、碧、妆、绿、丝、剪”等易错字
  • 积累春天词语：春风拂面、草长莺飞、万物复苏、柳绿花红
  • 会用“像、似的、好像”写比喻句
  • 区分同音或形近字：诗/寺、童/重、绿/录、植/值
常考题型
  • 根据诗句填空、连线诗人和题目
  • 阅读短文找春天景物，回答“从哪里看出春天来了”
  • 看图写春天，至少写清景物、颜色、动作和心情
  • 仿写句子：什么像什么，谁在哪里做什么

第二单元 阅读：关爱与劳动

复习检查

  • 阅读题先找原句

  • 注意糕、匹、叔等易错字

细化考点

课文理解

  • 《千人糕》能说清一块糕要经过哪些人的劳动

字词句

  • 积累词语：泥泞、年迈、晶莹、寻觅、特别、平平常常
  • 区分“的、地、得”的初步用法
  • 练习把长句读通顺，注意停顿
常考题型
  • 阅读题问人物品质时，用完整句回答
  • 排序题：按制作千人糕的过程排序
  • 联系生活写一件帮助别人或被别人帮助的事

常考题型

  • 根据偏旁猜字义，再查字典验证
  • 看拼音写词语，特别注意后鼻音和声调

第一单元 有余数的除法

平均分有余数的除法余数和除数的关系解决最多/至少问题

核心知识点

  • 余数一定比除数小
  • 商和余数都要有意义
  • 最多能装几份看商
  • 至少需要几份要看是否有余数

复习检查

  • 看到最多/至少先圈词
  • 写成 除法算式 + 单位 + 答
  • 有余数时判断要不要商加1

细化考点

概念和公式
  • 平均分后有剩余时，用有余数的除法表示
  • 算式格式：被除数÷除数=商……余数
  • 余数一定比除数小
  • 除数最小比余数大1
  • 被除数=除数×商+余数
常考题型
  • 最多能分几组：直接看商，余数表示剩下的
  • 至少需要几个盒子/几辆车：有余数时商要加1
  • 周期排列题：用总数÷周期，看余数确定第几个
  • 根据商和余数反推被除数
  • 判断竖式中商、余数、单位是否正确
易错检查
  • 余数不能等于或大于除数
  • 最多和至少不能混用
  • 答句单位要跟问题一致：人、条、盒、辆、次
  • 口算后也要写完整算式，不只写答案

第二单元 时、分、秒

认识时分秒钟面读数经过时间时间单位换算

核心知识点

  • 1时=60分
  • 1分=60秒
  • 分针走1大格是5分
  • 经过时间=结束时间-开始时间

复习检查

  • 钟面题先看时针在哪两个数之间
  • 经过时间题写起止时间
  • 不要把大格和小格混淆

细化考点

概念和进率
  • 1时=60分，1分=60秒
  • 分针走1小格是1分，走1大格是5分
  • 秒针走1小格是1秒，走1圈是60秒
  • 时针在两个数字之间，要读较小的那个整时
  • 半小时=30分，一刻钟=15分
常考题型
  • 看钟面写时间：先看时针，再看分针
  • 画钟面：先确定时针大概位置，再画分针
  • 经过时间：结束时间-开始时间
  • 开始时间：结束时间-经过时间
  • 结束时间：开始时间+经过时间
易错检查
  • 分针指向6是30分，不是6分
  • 分针走几个大格要乘5
  • 跨整点时先算到整点，再接着算
  • 单位换算先统一成分或秒再比较

第三单元 认识方向与位置

东南西北平面图方向路线描述

核心知识点

  • 上北下南左西右东
  • 先确定观察点
  • 路线题按顺序说方向和地点

复习检查

  • 题目问从哪里出发就以哪里为中心
  • 不要把左右和东西混在一起

细化考点

基础方向
  • 地图通常按上北、下南、左西、右东
  • 东和西相对，南和北相对
  • 东南、西南、东北、西北是斜方向
  • 先确定观察点，再判断方向
常考题型
  • 看平面图填某地在某地的什么方向
  • 按路线图写从起点到终点怎么走
  • 根据描述在图上标位置
  • 判断方向描述是否正确
易错检查
  • 问题问A在B的方向，就站在B看A
  • 不要把自己的左右当成地图方向
  • 路线题要按先后顺序写，不跳步

第四单元 认识万以内的数

计数单位数位顺序读数写数比较大小近似数

核心知识点

  • 从右往左个位、十位、百位、千位
  • 中间有0要读零，末尾0不读
  • 比较大小先看位数再从高位比

复习检查

  • 写数时数位要对齐
  • 近似数看接近哪个整十/整百/整千
  • 不要漏写0

细化考点

数位和读写
  • 从右往左依次是个位、十位、百位、千位
  • 10个一是十，10个十是一百，10个一百是一千
  • 读数从高位读起，中间有一个或两个0都只读一个零
  • 末尾的0不读，但写数时要占位
  • 算盘和计数器都要看清每一位上的珠子或数字
比较和近似
  • 比较大小先看位数，位数多的数大
  • 位数相同从最高位一位一位比
  • 近似数看接近整十、整百还是整千
  • 估算时先把数看成接近的整十或整百
易错检查
  • 写数不能漏0，例如三千零五要写3005
  • 读数时末尾0不读，中间0要读
  • 最大/最小组数题要看清数字能否重复

第五单元 分米和毫米

毫米分米米分米厘米毫米换算估测长度

核心知识点

  • 1厘米=10毫米
  • 1分米=10厘米
  • 1米=10分米
  • 先统一单位再计算

复习检查

  • 单位不同不能直接加减
  • 填单位题要联系生活实际
  • 答案后面必须写单位

细化考点

单位进率
  • 1厘米=10毫米
  • 1分米=10厘米
  • 1米=10分米=100厘米
  • 长度单位从小到大：毫米、厘米、分米、米、千米
  • 计算前先统一单位
常考题型
  • 填合适单位：铅笔长用厘米，硬币厚用毫米，课桌高用分米
  • 单位换算：几厘米=几十毫米，几分米=几十厘米
  • 比较长短：先换成同一单位
  • 画指定长度的线段，要从0刻度开始量
易错检查
  • 单位不同不能直接比较
  • 答案后面必须带单位
  • 估测题要联系真实物体大小
06

第六单元 两、三位数加减法

口算竖式计算进位退位验算估算

核心知识点

  • 相同数位对齐
  • 个位满十向十位进1
  • 不够减向前一位退1
  • 加减法可以互相验算

复习检查

  • 竖式不能省
  • 进位退位要标清
  • 计算后用估算看答案是否合理

细化考点

计算规则
  • 相同数位对齐，从个位算起
  • 个位满十向十位进1，十位满十向百位进1
  • 不够减向前一位退1，退1当10
  • 连续退位要特别标清
  • 加法用减法验算，减法用加法验算
常考题型
  • 直接写得数、列竖式计算、脱式或混合应用题
  • 估算结果是否合理
  • 根据表格或情境列加减法算式
  • 求比一个数多多少或少多少
易错检查
  • 竖式必须数位对齐
  • 进位的1和退位点不能忘
  • 横式答案要抄回去
  • 应用题最后写单位和答句
07

第七单元 角的初步认识

认识角直角锐角钝角图形中的角

核心知识点

  • 角有一个顶点两条边
  • 直角可以用三角尺判断
  • 角的大小和边张开的大小有关

复习检查

  • 数角时按顺序数
  • 不要把边长长短当成角大小

细化考点

图形概念
  • 角有一个顶点和两条边
  • 直角可以用三角尺的直角比一比
  • 比直角小的是锐角，比直角大的是钝角
  • 角的大小和两边张开的大小有关，和边的长短无关
常考题型
  • 判断图形中哪些是角
  • 数一个图形里有几个角或几个直角
  • 用三角尺画直角
  • 比较角的大小
易错检查
  • 开口不相连的不是角
  • 数角要按顺序，避免漏数或重复
  • 不要用边长判断角大小
08

第八单元 数据收集与整理

分类整理统计表读统计信息解决问题

核心知识点

  • 按同一标准分类
  • 统计表要看清项目和数量
  • 多多少少用减法

复习检查

  • 先看表头
  • 问题问哪个项目就找对应数据
  • 比较题要写清相差多少

细化考点

统计基础
  • 分类时只能按同一个标准整理
  • 统计表要看表头、项目和数量
  • 可以用画正字、打勾、数一数记录数据
  • 合计要把各项数量相加
常考题型
  • 根据统计表回答最多、最少、相差多少
  • 根据数据补全统计表
  • 根据统计结果提出建议
  • 读图表解决实际问题
易错检查
  • 最多最少不要看错项目
  • 相差多少用减法
  • 问题问几个就回答几个，不能只写数字无单位
