# 立夏期末复习特训工作台

这个仓库用于把二年级下学期语文、数学课本和试题照片整理成复习资料、错题集、打印 PDF 和静态网页。

## 隐私与版权规则

- 原始照片不提交到 GitHub。
- 孩子姓名、学校、班级、学号等信息需要在公开页面中脱敏。
- 课本原文的完整 OCR 结果只在本地保存；公开页面只发布复习摘要、错题改写和知识点大纲。

## 本地目录

```text
数学课本拍照/
数学试题拍照/
语文课本拍照/
语文试题拍照/
```
